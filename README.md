# Python-tkinter-libs
Tkinter向けライブラリ
Library for Tkinter

## 概要 Description
Tkinter向けライブラリ
Library for Tkinter


## 特徴 Features

- ScrolledFrameクラス  
	ScrolledFrame class
	- スクロールバーを持つフレームクラスを提供
		Provides Frame class with scrollbars  
	- マウスホイールでスクロール  
		Scroll with mouse wheel  

## 依存関係 Requirement

- Python 3.8.5  

## 使い方 Usage

```python
	from tkinter_libs import ScrolledFrame
	import tkinter as tk
	
	root = tk.Tk()
	scrolled_frame = ScrolledFrame(root)
```
